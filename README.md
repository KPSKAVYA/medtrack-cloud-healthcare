# MedTrack Cloud Healthcare System

MedTrack is a cloud-enabled healthcare management system built using Flask.

Features:
- User registration and login
- Doctor and patient dashboards
- Appointment booking
- Diagnosis submission
- Appointment search

Technologies Used:
- Flask
- AWS DynamoDB
- AWS SNS
- HTML/CSS
- Python

Deployment:
The application is deployed on AWS EC2 and uses DynamoDB as the database.
